== Description ==

WP Scaffolding is a complimentary plugin for working with the Fabric theme framework. WP Scaffolding helps in the creation of Custom Post Types and Custom Taxonomies by actually generating the required PHP files to register the post type or taxonomy, as well as any view files and controllers needed.

== Installation ==

Install and activate the plugin. This plugin is meant to be used in a local dev environment and requires that WordPress have write access the the active Fabric theme.
